# Automation Test Project

## 项目简介
基于 Python + Selenium + Pytest 搭建的自动化测试练习项目，包含：
- 电商系统 UI 自动化测试用例
- 接口测试脚本示例
- 测试报告与日志记录
- 基础的 Page Object Model 设计模式实践

## 技术栈
- 语言：Python 3.x
- UI 自动化：Selenium
- 测试框架：Pytest
- 报告工具：Allure Report

## 项目结构
- `tests/`：测试用例目录
- `pages/`：页面对象模型
- `utils/`：工具类（如配置读取、日志）
- `reports/`：测试报告输出目录
